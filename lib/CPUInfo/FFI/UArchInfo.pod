=head2 C<cpuid>

    my $cpuid = $uarchinfo->cpuid();

Returns an integer.

This is only available on C<x86_64>.

=head2 C<midr>

    my $midr = $uarchinfo->midr();

Returns an integer.

This is only available on ARM or ARM64.
